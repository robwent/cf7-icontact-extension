Simple way to integrate iContact mailing lists to Contact Form 7. Save your subscribers in a proper mailing list.

== Description ==

Integrate Contact Form 7 with [iContact](http://renzojohnson.com/contributions/contact-form-7-icontact-extension). Automatically add form submissions to predetermined lists in iContact, using its latest API. This iContact Extension supports multiple mailing lists and API Keys.

= Key Features =

* Easy to use
* Use a different mailing list per contact form
* Single opt-in - no confirmation email send to subscribers
* Double opt-in - confirmation email send to subscribers
* [Opt-in checkbox](http://renzojohnson.com/contributions/contact-form-7-icontact-extension/icontact-opt-in-checkbox) - add the chance to opt-in/opt-out
* Support for [custom fields](http://renzojohnson.com/contributions/contact-form-7-icontact-extension/icontact-custom-fields)
* Latest iContact API
* Constantly updated

= Requirements =

1. Self hosted WordPress.org installation (4.5 or higher)
2. Contact Form 7 (4.7 or higher)
3. iContact account




== Installation ==

1. Upload the entire '[contact-form-7-icontact-extension](http://renzojohnson.com/contributions/contact-form-7-icontact-extension)' folder to the '/wp-content/plugins/' directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Once the plugin is active, you will see a new 'tab' inside Contact Form 7 admin interface.

You will find '[iContact: Subscriber List Details](http://renzojohnson.com/contributions/contact-form-7-icontact-extension)' menu in your Contact Form 7 editor.

For basic usage, you can also have a look at the [Contact Form 7 iContact Extension homepage](http://renzojohnson.com/contributions/contact-form-7-icontact-extension), you can also have a look at the [Contact Form 7 homepage](http://contactform7.com).
